Kommandoer vi har lært:

git init - Setter opp en ny gitmappe
git status - Statusen til git
git add - Legg til filer i "indeksen"
git commit - Forteller git av vi skal huske alle disse filene.
git log - Viser en log over tidligere commits.
git log --graph - Viser en graf over tidligere commits. Se kul graf senere.
git branch [navn] - Lager en ny branch.
git switch [navn] - Bytter til en branch.
git switch -c [navn] - Lager og bytter til en branch.
git merge - Slå sammen en annen branch med nåværende.
git remote [navn] [addresse] - Lagrer en remote med gitt navn og gitt (typisk-github) addresse.
git push [remote] [branch] - Sender info til serveren gitt ved "remote"
